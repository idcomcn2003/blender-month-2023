# blender快捷键

## 单字母快捷键
~~~
F 填充面
X 删除
小键盘 7 顶视图
~~~

## 组合快捷键
~~~
Shift + A 添加物体
Shift + D 复制物体
Ctrl + 7 底视图
ALT + Z 透视显示模式
~~~

##　合并顶点
~~~
合并邻近的顶点: 选中几个非常靠近的顶点 右键,(或按M键),合并顶点|到中心
~~~

## 显示法线
~~~
编辑模式 -> 选择面 -> Mesh Edit Mode Overlays -> Normals
~~~

## 显示点线面
~~~
编辑模式 -> 1 切换到点
编辑模式 -> 2 切换到线
编辑模式 -> 3 切换到面
~~~
